# KAYKAYTALK

## 개요
socket을 이용하여 실시간 채팅 기능을 만들어보기.

## 기술 스택

### BackEnd
---
1. nestJS
    * Socket.IO
    * mongoDB (채팅 메시지 저장 및 조회)
    * MySQL (유저 정보저장)
    * mongoose
    * typeORM

### FrontEnd
1. React
    * 학습 후 추후에 정리할 예정

## 목표

웹 API는 **REST API**, **GraphQL** 이 두 버전을 사용하여 서버를 개발.

### 1차 목표
**REST API**는 12월 말까지 개발을 완료 할 예정.

### 2차 목표
**GraphQL API**는 1월 말까지 개발을 완료 할 예정.

### 3차 목표
화면은 3월 말까지 개발을 완료할 예정.
